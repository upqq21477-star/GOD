# 【頭七】遊戲企劃書
## ——有毒關係模擬器 × 動作 RPG × 催淚敘事

> **版本：** v0.1 概念企劃  
> **類型：** 動作 RPG / 敘事遊戲  
> **核心定位：** 把「有毒人際關係」做進即時戰鬥系統的高辨識度獨立遊戲
>
> 原戰棋設計，與遊戲核心不符，改成即時動作

---

## 📌 重要摘要

| 項目 | 內容 |
|---|---|
| **核心玩法** | 法師主角 + 三守護靈，九宮格隊形即時戰鬥 |
| **核心情感** | 一周目「爽」→ 二周目「碎」的雙層敘事 |
| **最強賣點** | 「情緒操作 = 戰鬥機制」，戰場即人際關係 |
| **競品參照** | 《Hi-Fi Rush》操作感、《Omori》情緒設計、《Undertale》後勁 |
| **目標受眾** | 有過有毒關係經驗的玩家、敘事遊戲愛好者、直播/實況族群 |
| **最大風險** | 隊友干擾頻率控制失當 → 玩家直接棄坑 |

---

## 一、世界觀與主題

### 1.1 題材背景

主角在頭七期間，以靈體形式與三位生前重要人物共存，展開對記憶與真相的最終探索。

### 1.2 雙層敘事結構

```
一周目（表層）：爽感體驗
  └─ 隊友各有性格，熱鬧又混亂，主角一帶三強行 Carry

二周目（底層）：真相解構
  └─ 所有「雷行為」皆有創傷來源
  └─ 玩家回頭重看一周目每個細節，情感崩潰
```

### 1.3 核心主題

> 「那些曾經保護你的人，是否真的選擇了你？」

---

## 二、角色設計

### 2.1 主角（法師）

- **定位：** 核心輸出，強大但容易誤傷隊友
- **機制特徵：** 法術帶有「友軍傷害（Friendly Fire）」，是觸發所有情緒事件的根源
- **設計邏輯：** 主角的「失控感」映射她在現實中對關係的「自責」

---

### 2.2 守護靈①——初戀（刺客）

| 項目 | 內容 |
|---|---|
| **戰鬥定位** | 擾敵、多重分身、高速攻擊 |
| **性格** | 話勞、垃圾話不斷、情緒化、刷存在感 |
| **前線行為** | 進入「驚嚇狀態」，攻速↑但命中率極低，亂打一通 |
| **被誤傷反應** | 氣泡遮半個螢幕，指控主角「妳是故意的吧！」 |

**雙層語義：**
- 一周目：煩人的麻雀，讓人放鬆警惕
- 二周目：話勞是害怕沉默的防衛機制；多重分身是生前被流言淹沒的殘影

**伏筆設計：**
當玩家靠近終局真相，她會**突然停止說話**。從最吵到完全沉默——這是最強的情緒核彈之一。

---

### 2.3 守護靈②——男友（戰士）

| 項目 | 內容 |
|---|---|
| **戰鬥定位** | 狂暴、純傷害、防禦上升 |
| **性格** | 無腦誇讚、情話轟炸、假溫柔 |
| **平常行為** | 拍馬屁、吹捧、情緒價值拉滿（讓玩家依賴他） |
| **危機行為** | 觸發【真愛撤離】，瞬移至畫面邊緣，留主角承受全部傷害 |

**關鍵機制——「可預測的人格缺陷」（非 RNG）：**

```
平常：
  誇讚 → 貼近 → 保護 → 情話 → 勝利時第一個衝來慶祝

Boss 高壓時，他的 UI 會出現預警訊號：
  呼吸急促 → 台詞變短 → 後退半步 → 笑容僵掉

若玩家沒有察覺預警 → 他在最後瞬間逃跑
```

**設計原則：**
- 逃跑機率**不是 RNG**，而是有跡可循的人格反應
- 玩家看到他逃時，應該罵的是「角色」，不是「設計師」
- **關鍵設計：** 前 10 次 Boss 戰他都沒逃，第 11 次玩家終於完全信任他——然後他退了

**雙層語義：**
- 一周目：垃圾男人
- 二周目：那是主角人生最後的創傷畫面——那個最愛說「有妳就安心」的人，在最後時刻先走了

---

### 2.4 守護靈③——有錢的爹（聖騎士）

| 項目 | 內容 |
|---|---|
| **戰鬥定位** | 守護盾、輔助 |
| **性格** | 爹味濃、斥責、命令、低頻壓迫 |
| **前線行為** | 【資產保全】觸發，自動 Lerp 後退，名言跳出：「前面是不可能站的，破產都不去！」 |
| **補給偏見** | 防禦盾**優先補給男友**，而非血量最低的主角 |

**位置邏輯：**
- 被放前線 → 退一格與主角並排
- 與主角同排 → 再自動退到後排
- 永遠不會主動站前線

**雙層語義：**
- 一周目：只顧女婿的控制狂老爹
- 二周目：盾牌是錢堆砌的安全感，也是一種監禁；老爹永遠後退這個畫面本身就是「疏離、衰老、金錢替代愛」的視覺符號，不需要任何文字說明

## 三、戰鬥系統設計

### 3.1 九宮格佈局

```
┌───┬───┬───┐
│   │   │   │  ← 前排（男友偽守護，高機率往後退）
├───┼───┼───┤
│   │ 主│   │  ← 中排（主角固定居中）
│   │ 角│   │
├───┼───┼───┤
│   │   │   │  ← 後排（老爹默認位置）
└───┴───┴───┘

初戀：側翼（嘲諷敵人，讓敵人不打主角）
男友：前方（名義上坦，實際會撤離）
老爹：後方（自動後退邏輯，資源壟斷）
```

### 3.2 隊形效果

| 隊形 | 效果 | 說明 |
|---|---|---|
| 進攻陣 | 守護靈全在主角前方 | 傷害最高，友傷風險最大 |
| 防禦陣 | 守護靈呈三角包圍主角 | 防禦提升，機動性下降 |
| 散亂陣 | 隊友各自跑位（被動觸發） | 高壓時隊友 AI 失控後的狀態 |

**視覺設計原則：**
- 守護靈平時維持高透明度/輪廓狀態，技能瞬間才完全實體化
- 移動時做「同步傾斜」：主角急轉，守護靈有微小延遲與傾斜，避免「四張貼圖平移」的廉價感

---

### 3.3 Swap 技能系統

**基礎規則：**
- 可 Swap 技能總計 **20 個**
- 每人（含主角）上限裝載 **3 個技能**
- 同一技能裝在不同角色身上，**效果完全不同**

**效果差異示例：**

| 技能 | 裝在主角 | 裝在初戀 | 裝在男友 | 裝在老爹 |
|---|---|---|---|---|
| 重擊 | 主角揮劍 | 分身追加一擊（自動）| 狂暴強化版 | 盾牌反擊 |
| 護盾 | 主角自護 | 分身誘敵盾 | 吸收後轉攻 | 範圍防護 |

**「運鏡技能」（建議 5 個）：**
按下後鏡頭強制拉近，三守護靈同時向中心轟炸大招——這是動漫爽感的核心演出。

**UI 原則：**
戰鬥中**不能換裝備**，只能換隊形。Swap 組合留在選單，戰鬥中僅提供 1-2 個快捷鍵觸發協作技。

---

### 3.4 煩躁值系統（Stress Bar）

**觸發來源：**
- 隊友逃跑
- 被誤傷後的情緒勒索
- 老爹斥責
- 初戀遮擋畫面

**效果設計（重要）：**

```
✅ 煩躁值應該影響：
  - 演出（主角動作更激烈）
  - 音效（背景音開始失真）
  - UI（輕微扭曲）
  - 台詞（守護靈說話頻率改變）

❌ 煩躁值不應直接大幅影響：
  - 操作精準度
  - 攻擊範圍（會讓玩家怪遊戲，不是怪隊友）
```

## 四、對話系統設計

### 4.1 氣泡對話（Popup Bubbles）

- 採用「動漫式氣泡」取代傳統對話框，於角色頭頂即時彈出
- **初戀：** 文字抖動，字體輕巧
- **老爹：** 文字巨大，字體粗重，有壓迫感
- **男友：** 流光效果，暖色系，虛偽的溫柔感

### 4.2 對話與玩家表現掛鉤

| 玩家表現 | 初戀 | 男友 | 老爹 |
|---|---|---|---|
| 打得很好 | 越來越嗨，廢話量飆升 | 誇讚密度提高 | 「哼，勉強及格。」|
| 打得很差 | 嘲諷但帶著擔心 | 假安慰 | 直接斥責 |
| 頻繁誤傷 | 爆發情緒勒索 | 開始後退、台詞變短 | 「這是哪門子法術！」|

### 4.3 對話的雙層語義（核心伏筆）

| 一周目台詞 | 二周目真相 |
|---|---|
| 初戀：「看我的分身，嚇死這隻小怪！」| 那些分身是當年流言蜚語的殘影 |
| 男友：「妳動手就好，剩下的交給我。」| 他從來沒有「剩下的」 |
| 老爹：「這張符多少錢知道嗎？別浪費！」| 所有的愛都被換算成了價格 |

## 五、敘事伏筆與反轉設計

### 5.1 一周目埋線清單

| 伏筆 | 一周目呈現 | 二周目真相 |
|---|---|---|
| 初戀的話勞 | 煩人的口頭禪 | 害怕沉默的孤獨感，源自校園霸凌 |
| 男友的誇讚 | 虛榮的情話 | 讓主角失去判斷力的毒藥 |
| 老爹的盾補給男友 | 爹味偏心 | 他真正在意的從來不是主角 |
| 男友逃跑 | 垃圾隊友 | 那是主角臨終時真實看見的畫面 |
| 老爹永遠後退 | 搞笑機制 | 疏離與衰老的視覺符號 |
| 初戀遮畫面 | 操作干擾 | 她的存在感來自恐懼，不是惡意 |

### 5.2 核心情緒核彈（按強度排序）

**核彈 1：初戀突然沉默**
```
前期：話勞到爆，一直洗畫面
某個特定時刻：完全停止說話
玩家反應：「她怎麼了？」→ 不舒服 → 查真相
設計原理：人類比起「吵」，更怕「突然安靜」
```

**核彈 2：男友第 11 次逃跑**
```
前 10 次 Boss：他沒逃，甚至偶爾真的擋了一刀
第 11 次：玩家完全信任他
Boss 最終技：他退了
玩家反應：「幹……我真的信過。」
```

**核彈 3：信物歸還儀式**
```
當玩家開始把奠儀、流言、紙錢歸還
守護靈逐漸停止說話
對話框內容變成破碎的雜音
從「熱鬧無腦」過渡到「死寂真相」
```

### 5.3 二周目的「偏見投影」轉折

玩家在二周目發現：

> 那些「雷行為」不是壞人行為，而是主角對他們最後的**記憶殘留**——
> 那個「會逃跑的垃圾男友」
> 那個「一直嘴的初戀」
> 那個「只顧女婿的老爹」
>
> 都是主角用自己的眼光，投射在他們靈體上的形象。

**這才是真正的主題：我們愛的、恨的、記得的——是真實的他們，還是我們想像的？**

## 六、開發注意事項

### 6.1 最大風險：玩家在真相前棄坑

**問題：** 隊友如果只有負面，玩家不會在二周目感到痛，只會覺得「這三個垃圾終於有理由了」——那不叫反轉。

**解法：70/30 原則**
```
70% 雷：逃跑、干擾、斥責、偏心
30% 真的救到：
  - 男友某次真的擋了一刀（即使只一次）
  - 初戀某次分身剛好保護了主角
  - 老爹的盾在最危急時出現
```

只有給過玩家「信任感」，背刺才會真正痛。

### 6.2 煩躁值的操控精準度警告

煩躁值**不能直接讓操作失控**，否則玩家會怪遊戲，沉浸感崩潰。應影響「演出層」而非「操作層」。

### 6.3 「VTuber 遊戲化」雙面刃

這款遊戲的混亂人格互動**極適合直播**，但也有觀看比遊玩更有趣的風險。
建議：確保「操作本身」提供玩家自主爽感，不能只靠「看角色互動」支撐體驗。

### 6.4 視覺遮擋問題

三守護靈 + 主角 + 特效 = 畫面極度擁擠。
解法：守護靈平時輪廓狀，技能瞬間才實體化閃現。

## 七、市場定位

### 7.1 競品參照

| 遊戲 | 借鑑方向 |
|---|---|
| 《Hi-Fi Rush》 | 動作節奏感、角色互動爽感 |
| 《Omori》 | 情緒設計、真相反轉的後勁 |
| 《Undertale》 | 行為有後果、玩家情感投射 |
| 《Persona》 | 人格就是機制 |
| 《Bojack Horseman》 | 有毒關係的真實刻畫 |

### 7.2 核心差異化

> **「把有毒關係做成即時戰鬥系統。」**

這件事在市場上幾乎沒有直接競品。

玩家記憶的不是技能 CD 或 DPS，而是：
- 「誰會怕」
- 「誰會逃」
- 「誰嘴硬但其實靠不住」

這會形成極強的**角色記憶**與**社群討論動能**（Meme、精華剪輯、實況效果）。

## 八、後續開發優先級建議

| 優先級 | 項目 | 原因 |
|---|---|---|
| ★★★ | 誤傷→情緒回應系統 | 核心差異化，戰鬥=人際關係 |
| ★★★ | 男友逃跑的「人格缺陷預警」機制 | 避免 RNG 感，建立角色可信度 |
| ★★★ | 守護靈沉默演出 | 最強情緒核彈，低開發成本高回報 |
| ★★ | 九宮格隊形切換 | 核心玩法，視覺辨識度高 |
| ★★ | Swap 技能效果差異 | Build 樂趣，提升重玩性 |
| ★ | 煩躁值系統 | 概念好但實作風險高，後期加入 |
